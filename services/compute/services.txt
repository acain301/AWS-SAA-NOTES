ami's are like docker images

key pair to ssh into instance

lifecycles, pending|running|stopping|stopped|shutting down|terminated

terminated is perminately deleted

bootstrap script can only be up to 16KB in size

ELB and autoscaling groups ... ELB can manage across multiple AZ's

Elastic IP's for a static ip

launch template, parameters for instances ... can be applied to autoscaling groups

cluster placement groups, cut down on latency

partition placement groups, instances on different hardware

puchasing options, ondemand|spot|savingsplan|reserved|dedicatedhosts|dedicatedinstances

reserved, 1 or 3 year committment to use compute power

savings, monthly discount

image builder, choose a base img and add/remove software you want

golden image, used across

***** Lightsail *****

quickest way to get an app up and running 

instances, containers, db's, storage, networking

can deploy ecr containers to lightsail 

***** ECS *****

fully managed container orchestration service that helps manage and scale containerized applications

containers run on EC2 or Fargate

ECS is proprietary that is a simple alternative to kubernetes and other orchestration applications


for EC2 deployment, you have to manage underlying infrastructure (docker containers, ECS agent, firewall, patch deployment) ... ECS will manage control plane

Fargate manages underlying infrastructure, serverless architecture

ECS Task Definition, blueprint for how containers should launch. (cpu, mem, ports, image, volumes)


ECS task is a running container that uses task definitions for it's settings

ECS Service, ensures that a certain number of tasks are running at all timese
